ภาพรวมวิชา

พื้นฐาน AI

ทำความเข้าใจ Large Language Models, Transformer Architecture และหลักการทำงานของ Generative AI

Prompt Engineering

AI Tools

ประยุกต์ใช้งาน

สร้างแอปพลิเคชัน AI ด้วย API, RAG System และ Workflow Automation ในสถานการณ์จริง
