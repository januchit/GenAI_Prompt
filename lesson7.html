7.1 AI กับการพัฒนาซอฟต์แวร์

AI กำลังเปลี่ยน Software Development อย่างรุนแรง จาก GitHub รายงานว่าผู้ใช้ Copilot เขียนโค้ด เร็วขึ้น 55% และรู้สึก "มีส่วนร่วม" กับงานมากขึ้น

AI ช่วยได้ในทุกขั้นตอนของ SDLC

Planning

วิเคราะห์ Requirements, สร้าง User Stories, ประเมิน Effort

Design

สร้าง Architecture, ERD, API Design, UI/UX Wireframe

Coding

Auto-complete, Generate Function, Refactor, Convert ภาษา

Testing

สร้าง Unit Test, Integration Test, Test Case จาก Requirements

Debug

วิเคราะห์ Error, หา Root Cause, เสนอวิธีแก้ไข

Docs

สร้าง README, API Docs, Comment, Changelog

ข้อมูลจาก GitHub (2024)
นักพัฒนาที่ใช้ Copilot เสร็จงาน เร็วขึ้น 55% | โค้ดที่ถูก Accept 46% มาจาก AI แนะนำ | 75% รู้สึกพอใจกับงานมากขึ้น

7.2 GitHub Copilot

GitHub Copilot github.com/copilot

AI Coding Assistant จาก GitHub + OpenAI รองรับ VS Code, JetBrains, Neovim, Visual Studio

ความสามารถหลัก

Inline Completion

แนะนำโค้ดบรรทัดหรือหลายบรรทัดแบบ Real-time ขณะพิมพ์ กด Tab เพื่อ Accept

Copilot Chat

คุยกับ AI เกี่ยวกับโค้ดได้ใน Sidebar ถามคำถาม, ขอ Explain, หรือขอ Fix

Inline Chat

กด Ctrl+I ใน Editor เพื่อสั่ง AI แก้ไขโค้ดที่เลือกไว้โดยตรง

Workspace Agent

ให้ AI อ่านทั้ง Project แล้วตอบคำถามเกี่ยวกับ Codebase ทั้งหมด

Slash Commands ใน Copilot Chat

/explainอธิบายโค้ดที่เลือก
/fixแก้ไข Bug ในโค้ดที่เลือก
/testsสร้าง Unit Tests
/docสร้าง Documentation Comment
/optimizeปรับปรุงประสิทธิภาพโค้ด
/newสร้างไฟล์หรือ Project ใหม่

เทคนิคใช้ Copilot ให้ได้ผลดีสุด

  • เขียน Comment ก่อน — พิมพ์ # Function to calculate compound interest แล้ว Enter Copilot จะสร้างโค้ดให้
  • ตั้งชื่อ Function ชัดเจน — def validate_thai_national_id(id_number): ดีกว่า def check(x):
  • ให้ Context ด้วยโค้ดที่มีอยู่ — Copilot ดู Context ของไฟล์ทั้งหมด เปิดไฟล์ที่เกี่ยวข้องไว้ด้วย
  • ใช้ Multiple Suggestions — กด Alt+[ และ Alt+] เพื่อเลือก Suggestion ที่ดีที่สุด

7.3 Cursor AI Editor

Cursor cursor.sh

Code Editor ที่ built-in AI (based on VS Code) ใช้ Claude + GPT-4 แก้ไขโค้ดทั้งไฟล์ได้ในคำสั่งเดียว

ฟีเจอร์เด่นที่แตกต่างจาก Copilot

Composer

สั่ง AI สร้างหรือแก้ไขโค้ดข้ามหลายไฟล์พร้อมกันในคำสั่งเดียว เช่น "สร้าง REST API สำหรับ User Management"

Chat with Codebase

AI อ่านทั้ง Codebase แล้วตอบคำถามเชิงลึก เช่น "ฟังก์ชันนี้ถูกเรียกจากที่ไหนบ้าง?"

@ References

อ้างอิงไฟล์, โฟลเดอร์, หรือ Documentation เช่น @file.py, @docs, @web

Apply Changes

AI เสนอการแก้ไขแบบ Diff แล้วคุณ Accept/Reject ได้ทีละส่วน ควบคุมได้ดี

7.4 Code Generation

การใช้ AI สร้างโค้ดต้องมีทักษะในการ "บอกความต้องการให้ชัดเจน" เพื่อให้ได้โค้ดที่ใช้งานได้จริง

ตัวอย่าง: สร้างฟังก์ชันจาก Natural Language

Prompt
สร้างฟังก์ชัน Python สำหรับตรวจสอบความถูกต้องของเลขบัตรประชาชนไทย 13 หลัก
โดยใช้ Algorithm ตรวจสอบ Checksum ตามมาตรฐาน กรมการปกครอง
คืนค่า True/False พร้อม Error Message ถ้าไม่ถูกต้อง
Python — AI Generated
def validate_thai_national_id(id_number: str) -> tuple[bool, str]:
    """
    ตรวจสอบความถูกต้องของเลขบัตรประชาชนไทย 13 หลัก
    Returns: (is_valid: bool, message: str)
    """
    # ลบช่องว่างและขีดกลาง
    id_clean = id_number.replace("-", "").replace(" ", "")

    if not id_clean.isdigit():
        return False, "เลขบัตรต้องเป็นตัวเลขเท่านั้น"

    if len(id_clean) != 13:
        return False, f"ต้องมี 13 หลัก (ได้รับ {len(id_clean)} หลัก)"

    # คำนวณ Checksum
    total = sum(
        int(id_clean[i]) * (13 - i)
        for i in range(12)
    )
    checksum = (11 - (total % 11)) % 10

    if checksum != int(id_clean[12]):
        return False, "Checksum ไม่ถูกต้อง เลขบัตรไม่ใช่ของจริง"

    return True, "เลขบัตรประชาชนถูกต้อง"

เทคนิค Code Generation ขั้นสูง

บอก Tech Stack ให้ชัด

"สร้าง REST API ด้วย FastAPI + SQLAlchemy + PostgreSQL รองรับ JWT Authentication"

ระบุ Edge Cases

"รองรับ Input ว่าง, ค่า null, และตัวเลขติดลบด้วย พร้อม Error Handling"

ขอ Refactor

"Refactor โค้ดนี้ให้ใช้ Pattern Repository, เพิ่ม Type Hints, และลด Duplication"

แปลงภาษา

"แปลง JavaScript function นี้เป็น Python โดยยังคง Logic เดิม และเพิ่ม Docstring"

7.5 Debugging ด้วย AI

AI ช่วย Debug ได้อย่างมีประสิทธิภาพ โดยวิเคราะห์ Error Message, Stack Trace และ Context ของโค้ด

วิธี Debug ที่ได้ผลดีสุด

1

แชร์ Error Message ครบถ้วน

Prompt Template
ฉันพบ Error ต่อไปนี้:
[วางตัว Error Message / Stack Trace ทั้งหมด]

โค้ดที่เกี่ยวข้อง:
[วางโค้ดที่ Error]

สิ่งที่ฉันพยายามทำ: [อธิบาย]
ช่วยวิเคราะห์สาเหตุและวิธีแก้ไข
2

บอก Environment

ระบุ Python version, Library version, OS เพราะบาง Bug เกิดจาก Version Conflict

3

ให้ AI อธิบายก่อน Fix

ขอให้ AI "อธิบายว่าทำไมถึง Error ก่อน แล้วค่อยแก้" จะช่วยให้คุณเข้าใจและเรียนรู้ด้วย

ตัวอย่าง Debug Session

Error ที่พบ
KeyError: 'user_id' Traceback: line 45, in get_user_data return data['user_id']
AI วิเคราะห์และแก้ไข
สาเหตุ: Dictionary `data` ไม่มี key 'user_id' เกิดจาก API Response มี field ชื่อต่างกัน

วิธีแก้:
# ใช้ .get() พร้อม default value
user_id = data.get('user_id') or data.get('id') or data.get('userId')
if not user_id:
    raise ValueError(f"ไม่พบ user_id ใน response: {list(data.keys())}")

7.6 Unit Testing & Code Review

สร้าง Unit Tests อัตโนมัติ

Prompt
สร้าง Unit Tests สำหรับฟังก์ชัน validate_thai_national_id() ด้วย pytest
ครอบคลุม: Happy Path, Invalid Format, Wrong Length, Wrong Checksum, Edge Cases
Python — pytest Generated
import pytest
from validator import validate_thai_national_id

class TestThaiNationalId:

    def test_valid_id(self):
        """Happy Path: เลขบัตรถูกต้อง"""
        valid, msg = validate_thai_national_id("1100600590167")
        assert valid == True

    def test_wrong_length(self):
        """ความยาวไม่ถูกต้อง"""
        valid, msg = validate_thai_national_id("12345")
        assert valid == False
        assert "13 หลัก" in msg

    def test_non_numeric(self):
        """มีตัวอักษรปนอยู่"""
        valid, msg = validate_thai_national_id("110060059016X")
        assert valid == False

    def test_wrong_checksum(self):
        """Checksum ไม่ถูกต้อง"""
        valid, msg = validate_thai_national_id("1100600590168")
        assert valid == False
        assert "Checksum" in msg

    def test_with_dashes(self):
        """รูปแบบที่มีขีด"""
        valid, msg = validate_thai_national_id("1-1006-00590-16-7")
        assert valid == True

AI Code Review

Security

Performance

Code Quality

"Review ตาม SOLID Principles, Clean Code, และ Best Practices สำหรับ Python"

Documentation

7.7 Prompt Patterns สำหรับ Coding

Pattern 1

The "Act As" Pattern

คุณเป็น Senior Python Developer ที่เชี่ยวชาญ Django REST Framework
Review โค้ดต่อไปนี้และแนะนำการปรับปรุง: [โค้ด]
Pattern 2

The "Step-by-Step" Pattern

สร้างระบบ Authentication ด้วย JWT โดยแบ่งเป็นขั้นตอน:
1. สร้าง User Model
2. สร้าง Register/Login Endpoint
3. สร้าง JWT Token Generation
4. สร้าง Auth Middleware
ทำทีละขั้นตอน อธิบายก่อนเขียนโค้ดแต่ละส่วน
Pattern 3

The "Template Method" Pattern

สร้าง CRUD API ตาม template นี้:
- Model: User {id, name, email, created_at}
- Endpoints: GET /users, POST /users, GET /users/{id}, PUT /users/{id}, DELETE /users/{id}
- Tech: FastAPI + SQLAlchemy + Pydantic
- รวม Error Handling และ Validation
Pattern 4

The "Rubber Duck" Pattern

ช่วยฉัน Debug โดยการถามคำถาม ไม่ต้องให้คำตอบทันที
ถามเกี่ยวกับ Logic ของโค้ดเพื่อช่วยให้ฉันหาปัญหาด้วยตัวเอง: [โค้ด]

7.8 สรุปและแบบฝึกหัด

สรุปสาระสำคัญ

  • AI ช่วยได้ทุกขั้นตอนของ SDLC ตั้งแต่ Planning, Design, Coding, Testing, Debugging, Docs
  • GitHub Copilot เด่นด้าน Inline Completion ใน IDE หลายตัว | Cursor เด่นด้าน Multi-file Editing
  • Code Generation ได้ผลดีเมื่อระบุ Tech Stack, Edge Cases และ Requirements ชัดเจน
  • Debug ด้วย AI — แชร์ Error Message ครบ, บอก Environment, ขอ Explanation ก่อน Fix
  • AI สร้าง Unit Test ครอบคลุมได้รวดเร็ว ลดเวลาเขียน Test ลงมาก

แบบฝึกหัดท้ายบท

1
ติดตั้งและทดลอง GitHub Copilot

ติดตั้ง GitHub Copilot ใน VS Code (ทดลองใช้ฟรี 30 วัน) เขียนฟังก์ชัน 5 ฟังก์ชันโดยเริ่มจาก Comment อธิบายก่อน สังเกตว่า Copilot แนะนำโค้ดอย่างไร

2
AI-Assisted Debugging

นำโค้ดที่เคย Debug ยากมา 1 ชิ้น (หรือสร้างโค้ดที่มี Bug ตั้งใจ) แล้วให้ AI ช่วย Debug พร้อมอธิบายสาเหตุ เปรียบเทียบกับการ Debug เอง

3
Generate Unit Tests

สร้างฟังก์ชัน Python อย่างน้อย 3 ฟังก์ชัน แล้วให้ AI สร้าง Unit Tests ที่ครอบคลุม Happy Path, Edge Cases และ Error Cases

4
Mini Project: AI-Assisted Web App

สร้าง Simple Web App (เช่น Todo List, Calculator, Currency Converter) โดยใช้ AI ช่วยตั้งแต่ต้นจนจบ บันทึก Prompt ที่ใช้ในแต่ละขั้นตอน
