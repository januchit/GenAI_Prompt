6.2 Diffusion Models — หัวใจของ Image AI

Diffusion Model คือ Neural Network ที่เรียนรู้ย้อนกลับกระบวนการ "เพิ่ม Noise" เพื่อสร้างภาพจาก Random Noise อย่างค่อยเป็นค่อยไป

Latent Diffusion Models (LDM)

Stable Diffusion และ DALL-E 3 ใช้ Latent Diffusion ที่ทำ Diffusion ใน Latent Space (พื้นที่มิติต่ำ) แทน Pixel Space ทำให้เร็วและใช้หน่วยความจำน้อยกว่ามาก

  • VAE (Variational Autoencoder) — แปลงภาพเป็น Latent Code และแปลงกลับ
  • U-Net / DiT — Neural Network ที่ทำ Denoising ใน Latent Space
  • CLIP / Text Encoder — แปลง Text Prompt เป็น Embedding เพื่อชี้นำการสร้างภาพ
  • CFG (Classifier-Free Guidance) — ควบคุมว่าภาพจะ "ตรงตาม Prompt" มากแค่ไหน

6.4 Midjourney

Midjourney midjourney.com

สร้างภาพคุณภาพ Aesthetic สูงสุด ใช้งานผ่าน Discord หรือ Web App (V6.1)

Commands สำคัญใน Midjourney

/imagine prompt: [your prompt]สร้างภาพจาก Prompt
--ar 16:9กำหนด Aspect Ratio (16:9, 4:3, 1:1)
--v 6.1เลือก Version โมเดล
--style rawลด Aesthetic อัตโนมัติ ควบคุมเองได้มากขึ้น
--q 2เพิ่ม Quality (ช้าลง แต่ละเอียดขึ้น)
--no [element]ห้ามมีสิ่งที่ระบุในภาพ Negative Prompt
--seed [number]กำหนด Seed เพื่อให้ภาพใกล้เคียงกัน
--sref [url]ใช้ภาพอ้างอิง Style (V6+)

6.5 Stable Diffusion

Stable Diffusion stability.ai

Open Source — รันบนเครื่องตัวเองได้ ปรับแต่งสูงสุด Fine-tune ด้วย LoRA/DreamBooth

วิธีใช้งาน Stable Diffusion

Online (ง่ายสุด)

ใช้ผ่าน Stability AI, Civitai, Leonardo AI ไม่ต้องติดตั้ง

Local (ควบคุมสูงสุด)

AUTOMATIC1111 WebUI หรือ ComfyUI ต้องการ GPU 6GB+

Google Colab (ฟรี GPU)

รัน WebUI บน Colab ฟรี ไม่ต้องใช้ GPU ตัวเอง เหมาะกับผู้เริ่มต้น

เทคนิคพิเศษของ Stable Diffusion

LoRA

Fine-tune โมเดลด้วยภาพ 20–50 รูป สร้างสไตล์หรือตัวละครเฉพาะตัว

ControlNet

ควบคุมท่าทาง, เส้นขอบ, ความลึก ด้วยภาพอ้างอิง ได้ผลแม่นยำสูงมาก

Inpainting

แก้ไขเฉพาะส่วนที่ต้องการในภาพ โดยไม่กระทบส่วนอื่น

Img2Img

ใช้ภาพเริ่มต้นแล้วให้ AI ปรับเปลี่ยนตาม Prompt ควบคุม Denoising Strength

Negative Prompt

ระบุสิ่งที่ไม่ต้องการในภาพ เช่น "blurry, deformed hands, watermark"

DreamBooth

Fine-tune โมเดลให้จำหน้าบุคคลหรือสินค้าเฉพาะ ใช้ภาพเพียง 5–10 รูป

6.6 Image Prompt Engineering

การเขียน Prompt สำหรับ Image AI ต่างจาก Text AI — ต้องบอก สิ่งที่อยู่ในภาพ, สไตล์, แสง, กล้อง, และอารมณ์ พร้อมกัน

โครงสร้าง Image Prompt ที่ดี

1
Subject
สิ่งหลักในภาพ "a majestic lion", "a Thai woman"
2
Environment
สภาพแวดล้อม "in a misty forest", "on a Bangkok street"
3
Action / Pose
ท่าทาง "running", "sitting gracefully", "looking at camera"
4
Lighting
แสง "golden hour", "dramatic rim lighting", "soft studio light"
5
Style
สไตล์ "photorealistic", "watercolor painting", "anime style"
6
Camera / Lens
กล้อง "shot on Canon 5D, 50mm, f/2.8", "wide angle", "macro"
7
Quality
คุณภาพ "8K", "highly detailed", "photorealistic", "masterpiece"

ตัวอย่าง Prompt เต็มรูปแบบ

Image Prompt ระดับ Professional
[Subject] An elderly Thai monk in saffron robes
[Environment] meditating inside an ancient temple in Chiang Mai at dawn
[Lighting] soft golden morning light streaming through ornate wooden windows
[Style] documentary photography, National Geographic style
[Camera] shot on Leica M11, 35mm f/2, shallow depth of field, film grain
[Quality] award winning photography, highly detailed, 8K resolution

Negative Prompt ที่แนะนำ (Universal)

worst quality, low quality, blurry, deformed, ugly, bad anatomy, extra fingers, watermark, signature, text, username, error, cropped, jpeg artifacts, out of frame

6.8 สรุปและแบบฝึกหัด

สรุปสาระสำคัญ

  • Diffusion Model เรียนรู้ย้อนกระบวนการ Noise เพื่อสร้างภาพจาก Random Noise
  • DALL-E 3 ง่ายที่สุด เข้าใจภาษาธรรมชาติดี | Midjourney คุณภาพ Aesthetic สูงสุด | SD ปรับแต่งได้มากสุด
  • Image Prompt ที่ดีต้องระบุ Subject, Environment, Lighting, Style, Camera, Quality
  • Negative Prompt ช่วยลดปัญหาภาพผิดรูปได้มาก
  • เทคนิคขั้นสูง: LoRA, ControlNet, Inpainting, Outpainting, Style Transfer

แบบฝึกหัดท้ายบท

1
เปรียบเทียบ 3 เครื่องมือ

สร้างภาพด้วย Prompt เดียวกันบน DALL-E 3 (ผ่าน ChatGPT), Midjourney และ Ideogram AI เปรียบเทียบคุณภาพและสไตล์

2
Image Prompt Workshop

เลือกสถานที่ในประเทศไทย 1 แห่ง เขียน Image Prompt ครบ 7 องค์ประกอบ สร้างภาพและปรับปรุงอย่างน้อย 3 รอบ

3
Style Transfer

นำภาพถ่ายของตัวเองหรือสถานที่ แล้วใช้ AI แปลงเป็นสไตล์ต่างๆ อย่างน้อย 3 สไตล์ (เช่น Anime, Oil Painting, Watercolor)

4
Mini Project: AI Art Gallery

สร้างภาพ AI 6 รูปในธีมเดียวกัน (เช่น "อนาคตของประเทศไทย" หรือ "เมืองใต้ทะเล") พร้อม Prompt ที่ใช้และคำอธิบายแต่ละรูป
